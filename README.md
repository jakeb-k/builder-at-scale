This is a Next.js 16 portfolio site for `jakebknowles.com`.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Deployment

Production deploys are handled through GitHub Actions and a static export served from the DigitalOcean droplet.

See [`docs/deployment.md`](docs/deployment.md) for the full GitHub and server setup.
